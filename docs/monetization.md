# Monetization

## Model: Freemium + Pro Subscription + Lifetime

### Free Tier

- Limited analyses per day (e.g., 5/day)
- Basic cipher detection (common encodings only)
- Coordinate format checker
- Field Quick Mode (limited uses)
- Local history (last 10 analyses)

### Pro Subscription — €7–10/month

- Unlimited analyses
- Full cipher/encoding library
- Advanced hint brainstormer (deeper pattern analysis)
- Priority processing
- Extended local history
- Early access to new features

### Lifetime — €39–49 (one-time)

- All Pro features, forever
- Available during early-access / launch period
- Rewards early adopters and builds initial cashflow
- May be retired or repriced after launch window
